We're funding MintCore (the CashTokens SDK) and CashMint Studio (the creator-first minting platform) to give builders a clean, well-documented, production-ready stack for CashTokens on Bitcoin Cash.

  • Goal: 25 BCH
  • Funding model: All-or-nothing via Flipstarter
  • Scope: MintCore SDK hardening, CashMint Studio MVP, docs, templates, and ecosystem onboarding

About the Builder

I'm Jeremy, a part‑time open‑source developer, father of seven, and long‑time Bitcoin Cash supporter. I've been building MintCore and CashMint Studio because I want BCH to be accessible to creators, developers, and new people entering the ecosystem.

Over the past year, I've been developing MintCore from the ground up as a clean, well‑tested CashTokens SDK designed for real‑world apps. Alongside it, I've been building CashMint Studio, a creator‑first minting platform focused on making token creation simple, safe, and approachable for beginners.

I'm committed to delivering a stable, documented, production‑ready stack that the BCH community can rely on, and I'm prepared to take this work full‑time if the funding and ecosystem support make it possible.

Why this matters

Right now, building with CashTokens means piecing together low-level tooling, re-reading forum posts and specs, and re-implementing the same logic in every project.

MintCore aims to be the standard CashTokens SDK for the BCH ecosystem. CashMint Studio turns that power into a creator-friendly minting experience for everyday users.

Together, they give:

  • Developers: a stable, well-tested SDK
  • Creators: a simple studio to mint, share, and monetize tokens
  • The ecosystem: a coherent, documented path from idea → token → real usage

What this Flipstarter funds

1. MintCore SDK hardening & releases

  • Finalize core APIs and semantics
  • Add test coverage and reference examples
  • Establish release lifecycle (tags, changelogs, versioning)
  • Write integration guides for wallets, apps, and services

2. CashMint Studio MVP

  • Creator flow: connect wallet → configure token → mint
  • Support for fungible tokens and basic NFTs
  • Clean, approachable UI with beginner-friendly copy
  • Basic analytics and status views for creators

3. Documentation, templates, and onboarding

  • “Start here” guides for developers and creators
  • Example repos and copy-paste templates
  • Clear contribution guidelines and issue templates

4. Ecosystem integrations & support

  • Work with wallets, explorers, and apps to integrate MintCore
  • Provide support and feedback via GitHub issues
  • Maintain a public roadmap and regular updates
